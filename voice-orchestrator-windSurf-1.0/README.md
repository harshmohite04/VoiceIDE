# Voice IDE Orchestrator

A voice-driven AI orchestration layer for code editors like Cursor and Windsurf.

## Features
- Voice command processing
- Natural language task decomposition
- IDE integration
- Task monitoring and status reporting
- Hands-free coding workflow

## Setup
1. Install dependencies: `pip install -r requirements.txt`
2. Configure your IDE API keys in `.env`
3. Run: `python main.py`

## Usage
- Speak naturally to give coding instructions
- Say "What's going on?" to check status
- Say "Stop" to cancel current operation
